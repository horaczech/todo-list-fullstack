# Fullstack TODO app

### using Express, React.js and Typescript

## Setup

```bash
git clone https://github.com/horaczech/todo-list-fullstack.git
cd todo-list-fullstack
npm run install
npm run watch
```

## Deploy

```bash
npm run install
npm run build
```
